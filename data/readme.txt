■はじめに

『VoiceroidUtil』は、VOICEROIDシリーズ等の文章読み上げソフトウェアを
自動操作して音声再生や音声ファイル保存を行う、音声素材作成支援ツールです。

また、AviUtl拡張編集ファイル(.exo)を作成させたり、作成されたファイルを
自動で『ゆっくりMovieMaker3』や『AviUtl拡張編集』に渡したりできます。

当ツールはすべてのVOICEROIDシリーズに対応しているわけではありません。
また、VOICEROIDシリーズ以外のソフトウェアについても一部対応しています。
対応ソフトウェアについては動作環境の項をご覧ください。

最新版は当ツール作者のWebサイトからどうぞ。
- https://ruche-home.net/

『VOICEROID+』および『VOICEROID2』は、
株式会社AHSから発売されている文章読み上げソフトウェアシリーズです。
動画投稿サイト向けの実況動画作成等に利用されています。
詳しくは公式サイトをご覧ください。
- https://www.ah-soft.com/

『ギャラ子Talk』は、ヤマハ株式会社から発売されている
『VOCALOID3 Library ギャラ子』に特典として同梱されている、
VOICEROID+と同等の機能を持つ文章読み上げソフトウェアです。
詳しくは公式サイトをご覧ください。
- https://www.vocaloid.com/products/show/v3l_galaco

『音街ウナTalk Ex』は、株式会社インターネットから発売されている、
VOICEROID+と同等の機能を持つ文章読み上げソフトウェアです。
詳しくは公式サイトをご覧ください。
- https://otomachiuna.jp/talkex/
- https://www.ssw.co.jp/products/talk/una/

『ガイノイドTalk』は、株式会社ガイノイドから発売されている、
VOICEROID2と同等の機能を持つ文章読み上げソフトウェアです。
詳しくは公式サイトをご覧ください。
- https://gynoid.co.jp/

『ゆっくりMovieMaker3』は、饅頭遣い氏が開発した動画作成補助ツールです。
音声ファイルを指定することで、ゆっくりボイス以外の音声にも対応可能です。
詳しくは公式サイトをご覧ください。
- https://manjubox.net/ymm3/

『AviUtl』は、ＫＥＮくん氏が開発した動画編集ツールです。
拡張編集Pluginを導入することで、タイムラインベースでの動画編集を行えます。
詳しくは公式サイトをご覧ください。
- http://spring-fragrance.mints.ne.jp/aviutl/

『ごちゃまぜドロップス』は、おおぶ氏が開発したAviUtlプラグインです。
AviUtl拡張編集ウィンドウへのファイルドロップを機能拡張することができます。
また、外部連携APIにより別アプリからの自動ファイルドロップを実現できます。
詳しくはGitHubリポジトリページをご覧ください。
- https://github.com/oov/aviutl_gcmzdrops

■動作環境

下記の環境が必要です。

- Windows 7 SP1, Windows 8.1, Windows 10
    - Windows 10 は May 2020 Update (version 2004) で確認しています。
    - Windows Vista SP2, Windows 8 でも恐らく動きますが保証しません。
    - Windows XP では動きません。

- .NET Framework 4.5 以降
    - Windows 8 以降であれば標準でインストール済みです。
    - Windows 7 SP1 の場合、 Windows Update よりインストールしてください。

- 文章読み上げソフトウェア
    - 現在は下記のもののみ対応しています。
      [ ] 内は動作確認済みバージョンです。
        - VOICEROID+ 結月ゆかり EX           [1.7.3]
        - VOICEROID+ 民安ともえ(弦巻マキ) EX [1.7.3]
        - VOICEROID+ 東北ずん子 EX           [1.7.3]
        - VOICEROID+ 東北きりたん EX         [1.7.4]
        - VOICEROID+ 琴葉茜・葵              [1.7.3]
        - VOICEROID+ 月読アイ EX             [1.7.3]
        - VOICEROID+ 月読ショウタ EX         [1.7.3]
        - VOICEROID+ 京町セイカ EX           [1.7.4]
        - VOICEROID+ 水奈瀬コウ EX           [1.7.3]
        - ギャラ子Talk                       [1.0.0]
        - 音街ウナTalk Ex                    [1.0.0]
        - VOICEROID2                         [2.0.5.0]
        - ガイノイドTalk                     [1.0.0.0]
    - VOICEROID2およびガイノイドTalkはエディターへの対応となるため、
      シリーズ製品を1つ以上インストール済みであれば利用可能です。

- ゆっくりMovieMaker3
    - 動作確認済みバージョンは 3.4.8.1 です。
    - 必須ではありません。無くとも動作します。

- AviUtl, 拡張編集Plugin, ごちゃまぜドロップス
    - 動作確認済みバージョンはそれぞれ 1.10, 0.92, 0.3.13 です。
    - 必須ではありません。無くとも動作します。
    - ごちゃまぜドロップスはPSDToolKitに同梱されているものでも構いません。
      https://github.com/oov/aviutl_psdtoolkit

■インストール手順

展開したファイル群を、フォルダ構成を崩さずに適当な場所に置いてください。
"VoiceroidUtil.exe" (結月ゆかりのアイコン)がツール本体です。

ツール本体のみのコピーでは動作しないためご注意ください。

■アンインストール手順

インストール時に配置したファイル群を削除してください。

一度でも使用すると、ローカルアプリケーションフォルダに設定が保存されます。
ローカルアプリケーションフォルダは Windows 7 以降の標準では次の場所になります。

- "C:\Users\[ユーザ名]\AppData\Local"

"AppData" フォルダは隠しフォルダです。
このフォルダ内の次のフォルダに設定が保存されます。

- "ruche-home\VoiceroidUtil"

完全に削除するには、この "VoiceroidUtil" フォルダも削除してください。
大したサイズにはならないはずなので、無理に削除する必要はありません。
